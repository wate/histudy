ボタン

本家

jQuery Mobileのボタンはモバイル端末上でボタンとして認識されやすいようなスタイル、また使いやすいよう最適化されています。
ボタンには大きく分けて２つの指定方法があります。
指定方法が違うというだけで見た目は同じです。

  • フォームボタン
  • リンクボタン

フォームボタンはbutton要素、input要素のsubmit,reset,button,imageのいずれかが指定されていればボタンとして自動的に整形されます。
リンクボタンはリンクに「data-role="button"」を指定することでボタンとして整形されます。

ボタンのレイアウト

ボタンには4つのレイアウト方法があります。 デフォルトは何も指定しないで表示した場合です。

  • ブロックレベル(Default)
  • インラインボタン
  • グループボタン
  • レイアウトグリッド

インラインボタン

jQuery Mobileのボタンはブロックレベル要素として親要素いっぱいに広がります。
ボタンはインライン要素として表示することもでき、文字列の量によってサイズを可変にすることもできます。
その場合は要素の属性に「data-inline="true"」を指定します。

はい いいえ わかりません

ソース

<a href="#" data-role="button" data-inline="true">はい</a>
<a href="#" data-role="button" data-inline="true">いいえ</a>
<a href="#" data-role="button" data-inline="true">わかりません</a>

グループボタン

グループボタンは複数のボタンを関連づけして表示したいときに利用します。
グループ化するには、ボタン群をコンテナとなる要素で囲みます。
その要素に対し「data-role="controlgroup"」を指定することで実現できます。

はい いいえ わかりません

ボタンを横向きに並べるには「data-type="horizontal"」を指定します。

はい いいえ わかりません

ソース

<div data-role="controlgroup">
	<a href="#" data-role="button">はい</a>
	<a href="#" data-role="button">いいえ</a>
	<a href="#" data-role="button">わかりません</a>
</div>

<div data-role="controlgroup" data-type="horizontal">
	<a href="#" data-role="button">はい</a>
	<a href="#" data-role="button">いいえ</a>
	<a href="#" data-role="button">わかりません</a>
</div>

レイアウトグリッド

レイアウトグリッドは、画面の横幅を指定の数だけ分割し、個々のグリッドのサイズに合わせたボタンを配置することができます。
レイアウトグリッドは最大5カラムまで対応でき、カラム数に合ったCSSクラスを指定します。ここでは詳細を省きます。
※グリッドレイアウトでボタンを配置した際、文字列に対してボタンの横幅が足りないと、三点リーダーで表示されます。

その他指定可能なボタンのdata属性

アイコンセット

jQuery Mobileでは、モバイルアプリでよく使うと思われるアイコンのセットを用意しています。
白いアイコンが黒い円に自動的に半透明になって重ねられ、背景と上手く組み合わせられます。
ボタンにアイコンを表示させたい場合は「data-icon=""」を指定します。
指定できるアイコンの種類は以下のようになっています。

arrow-r arrow-l arrow-u arrow-d plus minus back forward check delete search gear grid star refresh home info alert

アイコンの位置

初期状態では、アイコンはボタンの左側に表示されます。
「data-iconpos=""」指定することで、テキストの上下左右どこにでも表示させることが可能です。

left(左に表示)

←(Default)

right(右に表示)

→

↓

テーマ設定

jQuery Mobileは、ボタンのスタイルも自由に変更できるリッチなテーマシステムを備えています。 ボタンは手動で「data-theme="a-e"」を設定することで、コンテナとは異なった見え方にすることも出来ます。 jQuery Mobileはデフォルトで以下のテーマを持っています。

data-theme="a" data-theme="b" data-theme="c" data-theme="d" data-theme="e"

ソース

<a href="#" data-role="button" data-inline="true" data-theme="a">data-theme=&quot;a&quot;</a>
<a href="#" data-role="button" data-inline="true" data-theme="b">data-theme=&quot;b&quot;</a>
<a href="#" data-role="button" data-inline="true" data-theme="c">data-theme=&quot;c&quot;</a>

切り替え効果

jQuery Mobileでは、CSSによる6種類の切り替えエフェクトを用意しています。
これは、オブジェクトの切り替えやページ遷移、戻るボタンによる動作などで用いることが出来ます。

slide(Default) slideup slidedown pop fade flip

ソース

<a href="#dialog" data-role="button" data-inline="true" data-transition="slide">slide(Default)</a>
<a href="#dialog" data-role="button" data-inline="true" data-transition="slideup">slideup</a>
<a href="#dialog" data-role="button" data-inline="true" data-transition="slidedown">slidedown</a>
<a href="#dialog" data-role="button" data-inline="true" data-transition="pop">pop</a>
<a href="#dialog" data-role="button" data-inline="true" data-transition="fade">fade</a>
<a href="#dialog" data-role="button" data-inline="true" data-transition="flip">flip</a>
