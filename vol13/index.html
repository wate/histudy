ZenCodingって何？

一言で言えばHTMLやCSSをちょ～～簡単に書くことができるようになる魔法のツールです!
ZenCoding記法と記法を展開するためのショートカットを覚えると超高速でHTMLを書くことができます！
(※既にZenCoding自体の開発は終了して、Emmetという後継バージョンが出てます)

それ以外にも・・・

ZenCodingを使えば以下のようなことができます。
(※一部の機能を抜粋)

  • ZenCoding記法で包括
  • タグの外側や内側の内容を選択
  • 前/次の編集箇所への移動
  • 前/次のアイテムを選択
  • 対応する開始/終了タグに移動
  • 複数の要素を1行にする
  • 値の加算/減算(1|10|0.1)

ZenCodingの記法って
具体的にはどんなの？

タグと以下の内容を組み合わせてタグの構造を記述します

  • IDセレクタ：E#name
  • クラスセレクタ：E.name
  • 子供セレクタ：E>E
  • 兄弟隣接セレクタ：E+E
  • 属性セレクタ：E[attribute=value]
  • 繰り返し：＊<繰り返す回数>
  • 連番：$
  • タグ内テキスト：{}
  • グルーピング：()

まずは、基本から

  • div#name
  • div.name
  • div.one.two
  • head>link
  • table>tr>td
  • ul#name>li.item
  • p+p
  • div#name>p.one+p.two
  • p[title]
  • td[colspan=2]
  • span[title="Hello" rel]

これを展開すると・・・↓

<!-- 「div#name」を展開すると・・・ -->
<div id="name"></div>
<!-- 「div.name」を展開すると・・・ -->
<div class="name"></div>
<!-- 「div.one.two」を展開すると・・・ -->
<div class="one two"></div>
<!-- 「head>link」を展開すると・・・ -->
<head>
	<link rel="stylesheet" href="">
</head>
<!-- 「table>tr>td」を展開すると・・・ -->
<table>
	<tr>
		<td></td>
	</tr>
</table>
<!-- 「ul#name>li.item」 -->
<ul id="name">
	<li class="item"></li>
</ul>
<!-- 「p+p」を展開すると・・・ -->
<p></p>
<p></p>
<!-- 「div#name>p.one+p.two」を展開すると・・・ -->
<div id="name">
	<p class="one"></p>
	<p class="two"></p>
</div>
<!-- 「p[title]」を展開すると・・・ -->
<p title=""></p>
<!-- 「td[colspan=2]」を展開すると・・・ -->
<td colspan="2"></td>
<!-- 「span[title="Hello" rel]」を展開すると・・・ -->
<span title="Hello" rel=""></span>

さらに、こんなことも・・・・

  • p*3
  • p.title|e
  • p.name-$*3
  • select>option#item-$*3
  • ul+
  • table+
  • dl+

これを展開すると・・・↓

<!-- 「p*3」を展開すると・・・ -->
<p></p>
<p></p>
<p></p>
<!-- 「p.title|e」を展開すると・・・ -->
&lt;p class="title"&gt;&lt;/p&gt;
<!-- 「p.name-$*3」を展開すると・・・ -->
<p class="name-1"></p>
<p class="name-2"></p>
<p class="name-3"></p>
<!-- 「select>option#item-$*3」を展開すると・・・ -->
<select name="" id="">
	<option value="" id="item-1"></option>
	<option value="" id="item-2"></option>
	<option value="" id="item-3"></option>
</select>
<!-- 「ul+」を展開すると・・・ -->
<ul>
	<li></li>
</ul>
<!-- 「table+」を展開すると・・・ -->
<table>
	<tr>
		<td></td>
	</tr>
</table>
<!-- 「dl+」を展開すると・・・ -->
<dl>
	<dt></dt>
	<dd></dd>
</dl>

CSSもめっちゃ楽に・・・

全部紹介しきれないのでとりあえず一部を紹介

では、行きます！↓

まずは、margin

  • m  =>  margin:;
  • m:a  =>  margin:auto;
  • m:0  =>  margin:0;
  • m:2  =>  margin:0 0;
  • m:3  =>  margin:0 0 0;
  • m:4  =>  margin:0 0 0 0;
  • mt  =>  margin-top:;
  • mr  =>  margin-right:;
  • mb  =>  margin-bottom:;
  • ml  =>  margin-left:;

↓次↓

続いて、padding

  • p  =>  padding:;
  • p:0  =>  padding:0;
  • p:2  =>  padding:0 0;
  • p:3  =>  padding:0 0 0;
  • p:4  =>  padding:0 0 0 0;
  • pt  =>  padding-top:;
  • pr  =>  padding-right:;
  • pb  =>  padding-bottom:;
  • pl  =>  padding-left:;

↓次↓

続いて、text-align

  • ta  =>  text-align:;
  • ta:l  =>  text-align:left;
  • ta:c  =>  text-align:center;
  • ta:r  =>  text-align:right;
  • ta:j  =>  text-align:justify;

↓次↓

最後に、position

  • pos  =>  position:;
  • pos:s  =>  position:static;
  • pos:a  =>  position:absolute;
  • pos:r  =>  position:relative;
  • pos:f  =>  position:fixed;

※とりあえず入力がめんどくさそうなのを抜粋してますが、
これ以外にいろいろあります

ZenCodingやEmmetは省略記法のスニペットが

設定ファイルの場所は・・・↓

とりあえずわかる範囲で・・・

Mac持ってないから、Macに関してはわからん・・・・

  • Sublime Text 2
    C:\Users\(ユーザー名)\AppData\Roaming\Sublime Text 2\Packages\Emmet\emmet\snippets.json
  • Dreamweaver
    C:\Users\(ユーザー名)\AppData\Roaming\Adobe\Dreamweaver (バージョン)\ja_JP\Configuration\Commands\ZenCoding\my_zen_settings.js
  • Notepad++
    C:\Program Files\Notepad++\plugins\jN\includes\snippets.json
    ※64Bit版Windowsの場合は「Program Files」ではなく「Program Files (x86)」
  • Dreamweaver(ワテ改造版)
    C:\Users\(ユーザー名)\AppData\Roaming\Adobe\Dreamweaver (バージョン)\ja_JP\Configuration\Commands\ZenCoding\user.json
    (※メニューなどを日本語化してスニペットをJSONファイルで管理できるようにしたもの)
